# upload-file
### 简介

上传文件到FastDFS服务器，并对文件进行管理，前端用了layui。

### 前置条件

- 安装FastDFS，并且可以通过nginx代理访问
- 打包`fastdfs-client-java`项目
- 配置`upload`项目中的`application.properties`和`tracker.properties`
